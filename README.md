## SAPUI5 walkthrough
[Basic sapui5 tutorial](https://sap.github.io/ui5-tooling/pages/CLI/)


## Pre-requisites
* NodeJS
* [UI5 cli](https://sap.github.io/ui5-tooling/pages/CLI/)

1. Install dependencies
```sh
npm install
```

2. Run server
```sh
ui5 serve
```
